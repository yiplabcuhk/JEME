The programs can be run on Linux (tested on version 3.10.0-327.3.1.el7.x86_64).
The R scripts were tested on R version 3.2.0. The package glmnet needs to be installed for learning LASSO models.
Part 2 of JEME involves applying a model in Weka model format. The programs were tested on Weka version 3.7.13.
